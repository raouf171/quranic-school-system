#prequisties :

### For Backend (Laravel)
- **PHP** 8.1 or higher ([Download](https://www.php.net/downloads))
- **Composer** ([Get Composer](https://getcomposer.org/))
- **MySQL** or MariaDB ([Download](https://www.mysql.com/downloads/))
- **Postman** (optional, for testing APIs) ([Download](https://www.postman.com/))

### For Frontend (Android)
- **Android Studio** (latest stable) ([Download](https://developer.android.com/studio))
- **Android SDK** (API level 24 or higher)
- **JDK** 11 or higher (bundled with Android Studio)

### For Version Control
- **Git** ([Download](https://git-scm.com/))
- **GitHub account** (invited as collaborator)
